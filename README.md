# myportfolio-site

大学生・ブロガー **tabunnhito** のポートフォリオサイトです。
ビルド不要の静的サイト（HTML / CSS / JS）で、GitHub Pages 等にそのまま公開できます。

## ファイル構成

- `index.html` — サイト本体（1ページ構成）
- `style.css` — スタイル
- `script.js` — ナビ開閉・フェードイン・フォーム送信

## ローカルで確認する

`index.html` をブラウザで直接開くだけで表示できます。

## 公開前に差し替える項目

- X（Twitter）のプロフィールURL
